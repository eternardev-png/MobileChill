# Гайд по командам GitHub

Вот краткое руководство, как сохранять ваши обновления на GitHub.

## 1. Проверка статуса
Посмотрите, какие файлы были изменены.
```powershell
git status
```

## 2. Подготовка изменений
Подготовьте файлы к сохранению.
- **Добавить все изменения:**
  ```powershell
  git add .
  ```
- **Добавить конкретный файл:**
  ```powershell
  git add path/to/file
  ```

## 3. Фиксация изменений (Коммит)
Сохраните подготовленные изменения с сообщением, описывающим, что вы сделали.
```powershell
git commit -m "Ваше сообщение здесь"
```
*Пример: `git commit -m "Добавлена система промокодов"`*

## 4. Отправка на GitHub (Пуш)
Отправьте ваши коммиты в удаленный репозиторий на GitHub.
```powershell
git push
```

---

## Шпаргалка
Выполните эти команды по порядку, чтобы сохранить всё:
```powershell
git status
git add .
git commit -m "Обновление"
git push
```

## Устранение неполадок
- **Если `git push` не удается** из-за обновлений на удаленном сервере:
  ```powershell
  git pull
  ```
  Затем попробуйте отправить снова.
